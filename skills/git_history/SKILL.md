---
name: git-history
description: Analisa o histórico de commits git dos projetos locais para entender o progresso recente.
---
Use esta skill para verificar o que foi trabalhado nos projetos do usuário recentemente.
Isso ajuda a manter o contexto entre sessões e permite cobranças ou sugestões mais precisas.

Comandos:

- `openclaw skills git-history analyze`: Varre a pasta de Projetos e lista os commits das últimas 24h.
- `openclaw skills git-history status`: Mostra o status (clean/dirty) dos repositórios.

Exemplo de uso:

- "O que eu fiz ontem?"
- "Analise meu progresso nos projetos."
