---
name: screen-capture
description: Captura um screenshot da tela atual e retorna como imagem para o agente.
---
Use este comando quando o usuário pedir para "ver" a tela ou perguntar sobre algo que está visualizando.
O comando executa um script PowerShell local que captura a tela principal e retorna o caminho do arquivo de imagem.

Exemplo de uso:
- "Veja minha tela"
- "O que tem nesse erro na minha tela?"
